

# 🔍 Nmap Network Scanning & Vulnerability Assessment

## 📌 Project Overview
This project demonstrates network reconnaissance and vulnerability analysis using Nmap on Kali Linux in a controlled environment.

## 🎯 Objectives
- Identify open ports
- Detect running services
- Discover potential vulnerabilities

## 🛠 Tools Used
- Nmap 7.95
- Kali Linux

## 🔎 Scan Techniques
- SYN Scan (-sS)
- Service Version Detection (-sV)
- Aggressive Scan (-A)
- Full Port Scan (-p-)
- NSE Vulnerability Scan

## 📂 Project Files
- full_scan.txt → Complete scan results
- vuln_scan.txt → Vulnerability findings
- report.md → Detailed analysis

## ⚠️ Key Findings
- Open ports identified
- Services exposed to network
- Potential vulnerabilities detected

## ✅ Recommendations
- Disable unused services
- Apply security patches
- Configure firewall rules

## 💡 Skills Demonstrated
- Network scanning
- Enumeration
- Vulnerability analysis
- Report writing
